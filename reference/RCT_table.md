# Fictional Results from a Workers' Randomized Control Trial

Fictional Results from a Workers' Randomized Control Trial

## Format

A 2-by-2 table, with a *column* for each group and a *row* for the
diagnosis.

    data("RCT_table")
    RCT_table
    #>            Group
    #> Diagnosis   Treatment Control
    #>   Sick             71      30
    #>   Recovered        50     100
